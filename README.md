# E-commerce-Sales-and-Delivery-Analysis-SQL-Power-BI
## Project Description
This project analyzes an e-commerce dataset using SQL and Power BI. 
The process included data cleaning, relational database modeling, data import into Power BI, DAX measures creation and dashboard development for sales and logistics analysis.

## Tools Used
- SQL
- Power BI
- DAX
- Excel

## Project Process
1. Data cleaning in SQL
2. Relational data modeling
3. Data import into Power BI
4. Creation of DAX measures
5. Sales dashboard development
6. Logistics and delivery analysis

## Analysis Includes
- Total sales
- Average ticket
- Sales by month
- Top 10 products
- Sales by payment type
- Orders delivered vs delayed
- Delivery time analysis

## Dashboards
Sales and Logistics dashboards were developed in Power BI to analyze commercial performance and delivery performance.
