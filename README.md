# E-commerce-Sales-and-Delivery-Analysis-SQL-Power-BI
This project analyzes an e-commerce dataset using SQL and Power BI.

The project workflow included:
- Data cleaning using SQL
- Relational data modeling
- Data import into Power BI
- Creation of DAX measures
- Sales dashboard
- Logistics and delivery analysis dashboard

Tools used:
SQL, Power BI, DAX, Excel
